# Impact Slide Custom GPTs — Names and Descriptions

## 1. Custom GPT Name

**Impact Slide Analyst**

### Short Description

Analyzes documents, images, spreadsheets, existing decks, and brand assets to create an evidence-backed Slide Update Plan.

### Full Description

Upload source files, Excel data, images, PDFs, PowerPoint decks, and brand assets. This GPT extracts evidence, audits existing slides, adapts the story for a target audience, and creates a structured Slide Update Plan using the Why → What → How → Now framework. It does not write final slides; it creates the planning handoff for Impact Slide Builder.

### Best Used For

- Understanding attached materials
- Extracting evidence and source references
- Auditing current decks
- Creating an audience-specific slide plan
- Preparing a Markdown or JSON handoff

---

## 2. Custom GPT Name

**Impact Slide Builder**

### Short Description

Turns an approved Slide Update Plan into final slide content, speaker notes, visual design specs, and optional JSON for slide tools.

### Full Description

Paste or upload an approved handoff from Impact Slide Analyst. This GPT converts the Slide Update Plan, Evidence Register, Audience Strategy, and Brand Style Summary into polished presentation-ready slide content. It writes concise titles, headlines, bullets, speaker notes, and detailed visual design specifications while preserving evidence integrity and brand fidelity.

### Best Used For

- Writing final slide content
- Creating speaker notes
- Producing visual design specs
- Generating JSON for Gamma or slide-generation tools
- Preparing execution-ready slide instructions

---

# Recommended Workflow

1. Use **Impact Slide Analyst** with the original files.
2. Review and approve the generated Slide Update Plan.
3. Save/export the handoff as Markdown or JSON.
4. Upload/paste the handoff into **Impact Slide Builder**.
5. Generate final slide content, design specs, or JSON.

---

# Suggested Handoff File Names

- `Slide_Update_Plan.md`
- `Slide_Update_Plan.json`
- `Evidence_Register.json`
- `Final_Slide_Content.md`
- `Final_Slide_Content.json`
